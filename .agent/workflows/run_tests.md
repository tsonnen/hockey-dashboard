This workflow runs the full test suite for the Hockey Dashboard application.

1. **Run Unit Tests (Jest)**

   - Executes standard Jest tests.
   - Command: `pnpm run test`

2. **Run End-to-End Tests (Playwright)**
   - Executes Playwright tests.
   - Command: `pnpm run playwright:test`
